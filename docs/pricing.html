Frequently asked questions

Is there a free plan?

Yes. Free Cloud is a free cloud plan for up to 5 users with a lightweight tracker set of 8 plugins. The full product lives in Cloud: a 14-day trial with no credit card. For on-prem, a pilot period is possible — write to us.

How do plugin sets differ between plans?

Free Cloud ships a lightweight tracker set: search, calendar, checklists, reports, dashboards, notifications, time tracking and archive. Every paid plan includes all 24 plugins — Service Desk with the portal, SLA, automation and the MCP server.

Cloud or on-prem — which one to pick?

Cloud gets you started in minutes, from $5 per user per month with yearly billing, with updates and backups on our side. On-prem keeps data inside your perimeter at a flat price per installation: from about 25 users Team already beats the cloud, and Unlimited works out to about $2 per user per month at 200 users.

What does the on-prem price include?

Team is $1,250 per year: up to 25 users, one installation, updates included. Unlimited is $5,000 per year: unlimited users. Setup is a separate service: basic setup is a one-time $1,100, an implementation with process configuration and training starts at $1,900.

Can we migrate from Jira?

Yes. The built-in import migrates projects from a Jira JSON backup with field, status and priority mapping. Enterprise includes turnkey migration done by our team.

What is needed for an on-prem installation?

A server with Docker: about 2 GB of RAM for the compact mode and about 4 GB for the split production mode. We install and configure everything as part of the setup.

How do I get support?

Email support is included in every paid plan. A support subscription starting at $380 per month adds consulting and a response-time SLA; Enterprise adds a 4-hour SLA and a dedicated channel. You can reach us via the contact page.
